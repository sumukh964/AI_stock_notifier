Project overview

This is a production-grade AI pre-market stock intelligence system that runs fully automated on GitHub Actions every day at 7:30 AM IST before NSE opens.

It scrapes financial headlines from Economic Times, Moneycontrol, and LiveMint, runs each headline through FinBERT — a transformer model fine-tuned on financial text — and aggregates signals per company to send only the highest-confidence BUY alerts to Telegram.

A market mood filter suppresses all BUY signals when the overall headline sentiment is bearish, dramatically reducing false alarms.

Tech stack
Python 3.10 FinBERT HuggingFace Transformers spaCy NER BeautifulSoup4 GitHub Actions Telegram Bot API PyTorch
System pipeline
01
News Collection
Scrape ET, Moneycontrol, LiveMint · top 30 unique headlines
02
FinBERT Inference
Sentiment + softmax confidence per headline · torch.no_grad()
03
spaCy NER Detection
ORG entity extraction · NSE whitelist · noise filter
04
Multi-news Aggregation
Majority vote + avg confidence per company across all headlines
05
Market Mood Filter
>50% negative headlines → Bearish → suppress all BUY signals
06
Telegram Delivery
One alert per BUY company · secure env-var credentials
